The Query User's Guide is considered a 'living' document, and is intended to grow and evolve as more and more users write queries and identify ‘hints’ to be included. Any additions identified by end users should be sent to the MESA Mailbox or the MESA Webmaster to be considered for inclusion.

EDW MESA Query User’s Guide The EDW MESA Query User’s Guide provides information about the basic MESA tables, standard selection criteria, and source specific variations & exceptions to assist End Users using SQL to write queries and reports against EDW. The Query User's Guide is considered a 'living' document, and is intended to grow and evolve as more and more users write queries and identify ‘hints’ to be included. Any additions identified by end users should be sent to the MESA Mailbox or the MESA Webmaster to be considered for inclusion.
