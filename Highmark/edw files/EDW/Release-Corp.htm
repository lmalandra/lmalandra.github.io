EDW Corporate Release

The links below provide the most up to date information for the Enterprise Data Warehouse Release.  The information is usually updated by the end of the week, unless required sooner.
  • Contents – EPN report that lists project names, descriptions and contacts.
  • Milestones – Excel report that summarizes key milestone dates (timeline) for corporate releases.
  • Data Model Build Plan - Excel document that summarizes the key data model dates for the release.
  • LST Calendar - Represents a summary of key EDW release milestones such as OSCAR Adjudication code drop dates, OSCAR LST dates, EDW HPLC lifecycle dates, etc. Check back periodically as dates are subject to change when new information becomes available. The 'last updated' date is included on the calendar for reference purposes. See separate 'Data Model Build Plan' for EDW Build dates. The LST dates are only as current as of the last update date noted on the calendar. See OSCAR for most up to date information regarding LSTs.
  • Project Plan – Represents the detailed work plan for the release including estimated timeframes and progress by each project phase.
  • Implementation Plan – This is the detailed plan for implementation.  It starts a few weeks prior to the release and ends a few days after the release.
