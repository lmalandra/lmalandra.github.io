EDW Loads by System ID
The term "System ID" is used by the EDW to denote an inbound source. For example, within the Medical Experience Subject Area "COM" signifies the Complementary source of data. This report shows the earliest load of each System ID and the latest load of System ID into the EDW. Currently, this report is updated on a monthly basis; however, in the future, this report will be updated on a daily basis.
