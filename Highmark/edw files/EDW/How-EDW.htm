How do I access the EDW?

Security Access Request Application
Before you are able to utilize the EDW, you must request access by submitting a SARA form. SARA (Security Access Request Application) was developed with the goal of providing ease of use for the requestor. The initial entry screens allow you to submit a LID or Application ID for which you are requesting access, or you can request access for multiple LIDs by cost center or job title.

Click here to access SARA.

For your convenience, we have provided a specific set of instructions for each major EDW subject area.
  • End User Access Layer
  • UCCI Dental Experience
  • Clinical Member Reporting Repository
  • All Other EDW Subject Areas
Questions regarding the overall SARA entry screens or applications for which you are trying to request access should be sent to the SARAinquiry mailbox.

Security Overview Diagram
The Security Overview Diagram provides an overview of the EDW design. The high-level process is to extract Source Data, move it to an EDW Staging area to perform transformations such as assigning DRG code, and then load the EDW Base database. The data is then available for access by End Users and Business applications.

The security components and corresponding control points in the EDW MESA design are identified as follows:
  • End User security
  • Third Party Tool security
  • Downstream Application security
  • MESA ISG Support Area security
