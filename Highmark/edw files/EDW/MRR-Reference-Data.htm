MRR Reference Data

Below please find the most current reference data for MRR. Reference Code Tables (RCTs) were not built in EDW for MRR data. MRR data is stored in a denormalized model. MRR records (rows) contain codes and corresponding descriptions within the same row.

Reference data is available for DXCG/ID & Strat/Risk Score, C3 and MaxMC sources.
