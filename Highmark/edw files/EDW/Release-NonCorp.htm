Non-Corporate Release

The links below provide the most up to date information for the Enterprise Data Warehouse Release.  The information is usually updated by the end of the week, unless required sooner.
  • Contents – EPN report that lists project names, descriptions and contacts. The Release Contents report includes both Upcoming Non-Corporate and Project Specific Non-Corporate releases.
  • Project Plan – Represents the detailed work plan for the Upcoming Non-Corporate release including estimated timeframes and progress by each project phase.
  • Implementation Plan - This is the detailed plan for the Upcoming Non-Corporate releases implementation only.  It starts a few days prior to the release and ends a few days after the release. Project Specific Release Implementation Plan can be found in the Non-Corporate release folder on the shared drive.
Detailed release information will be available according to the following timeline:
  • Contents – 3 months prior to the release
  • Data Model - 3 months prior to the release
  • Project Plan - 3 months prior to the release
  • Implementation Plan - 1 month prior to the release
