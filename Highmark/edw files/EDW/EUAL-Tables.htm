EUAL Tables

Claim Tables
The EUAL table rows on the following page are all at the Claim Line level. The key that should be used to join the Claim related tables within EUAL is the Medical Expense Event ID from the EDW (EAMEE_ID) and the Claim Line Number blind key from EDW (on all EUAL Claim related tables this is the LINE_BLIND_KEY). The "blind key" Claim Line value (LINE_BLIND_KEY) is the recommended criteria when joining EUAL tables, in order for maximum query efficiency. However, there is also a column on each EUAL table that houses the actual Claim Line number value, which is EACL_NO.
*See Appendix-B for a detailed EUAL Data Dictionary, listing all columns available in the EUAL Claim tables.

Other EUAL Tables
Unlike the Claim Tables, these EUAL table rows are not at the Claim Line level, and therefore cannot be joined with other EUAL tables.
